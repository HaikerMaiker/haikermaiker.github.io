I made this webpage to showcase some of the projects I've been working through the years. Below I leave some important hyperlinks.

Curriculum

Academic Project Isometric Sokoban

Isometric "Sokoban" style game, made as the final assignment for Harvard's CS50x using Lua and the Love framework. I developed a rendering sytem that transforms and renders strings into isometric levels displayed on the screen, plus the logic system for basic movement, block pushing, switches and piston functionality.

Download

Professional project PLC Simulator

Didactic simulator developed for Android using Unity 3D, C#, Firebase and Plastic SCM. I worked as part of a team to deliver updates to the public while meeting deadlines. This updates consisted of game optimizations and new functionalities for this didactic game.

Google Play

Professional project Cubam

2D boss-rush game prototype created for a Fiverr client. I implemented movement, atacks and abilities for bothe the PC and the enemies according to the client's specifications. The game's main mechanic is the ability to rewind time mid-fight at command.
